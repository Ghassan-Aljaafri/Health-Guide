## Configuration Steps

>copy .env.example file to .env
>
>create new Database
>
>edit .env file :point_down:
>
>
>DB_DATABASE=
>

1) ```$ composer install```
2) ```$ php artisan key:generate```
3) ```$ php artisan jwt:secret```
4) ```$ php artisan migrate --seed```
